### 🔭 Projects

#### 🎯 Perimeter Pulse: Context-Aware Stress Monitoring System
Real-time workplace analytics system processing RTSP CCTV streams for emotion and posture analysis.
- Mini-Xception facial emotion recognition — **89% accuracy at 30 FPS**
- BlazePose-based posture analysis — **35 FPS, 33 body key points**
- Privacy-first design: k-anonymity, encrypted identity mapping, GDPR-aligned consent
- Validated in a 40-participant study with **75% acceptance rate**
- 🏆 Best Paper Award, IEEE ICWITE 2025 · Published on IEEE Xplore

`Python` `TensorFlow` `BlazePose` `RTSP` `AWS S3` `NumPy`
